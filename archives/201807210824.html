    获取和创建项目

    • git init

    • 创建新的git仓库

    • git clone

    • 拷贝一个git仓库到本地

    快照

    • git add

      • 添加文件到缓存

    • git status

      • 查看提交状态

    • git diff

      • 查询缓存中和当前文件的区别

    • git commit

      • 将缓存内容提交到本地仓库

    • git reset HEAD

      • 取消已经缓存的内容

    • git rm

      • 从工作目录中手工删除文件

    • git mv

      • 从工作目录中移动，重命名文件
