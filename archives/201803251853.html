    Python循环控制

    程序在一般情况下是按顺序执行的，但编程语言也提供了各种控制结构，允许更复杂的执行路径。循环语句允许我们执行一个语句或语句组多次。

    while循环

    在给定的判断条件为 true 时执行循环体，否则退出循环体。

    eg.输出0到4的数字

    count = 0
while (count < 5):
   print 'The count is:', count
   count = count + 1

    for循环

    重复执行语句

    eg.输出python的每一个字母

    for letter in 'Python': 
   print '当前字母 :', letter

    循环嵌套

    可以在循环体(while/for)中嵌套循环(while/for)

    eg.嵌套循环输出2~100之间的素数

    i = 2
while(i < 100):
   j = 2
   while(j <= (i/j)):
      if not(i%j): break
      j = j + 1
   if (j > i/j) : print i, " 是素数"
   i = i + 1
