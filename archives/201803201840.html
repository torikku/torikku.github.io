    Python条件控制

    Python条件语句是通过一条或多条语句的执行结果（True或者False）来决定执行的代码块。

    if...else...语句

    eg. 

    a = input('a:')
b = input('b:')
if(a>b):
    print a,'>',b
else:
    print a,'<',b

    if...elif...else...语句

    eg.

    score = raw_input('score:')
score = int(score)
if(score >= 90) and (score <= 100):
    print 'A'
elif(score >= 80):
    print 'B'
elif(score >= 60):
    print 'C'
else:
    print 'D'

    if语句嵌套

    eg.

    x = -1 
y = 99
if(x >= 0): 
    if(x > 0): # 嵌套的if语句
        y = 1
    else:
        y = 0

    y = -1
