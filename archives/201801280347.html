    博客回顾－－－安装主题

    之前已经写了怎么把一个基本的站点搞出来了，但是基本就是一个页面，什么都没有，甚至主题也没有，接下来讲下主题的安装

    这里以主题bootstrap2为例，同样还在myBlog目录下：

    //进入到目录
cd myBlog
//获取主题git
git clone https://github.com/getpelican/pelican-themes.git
//进入下载到主题目录
cd pelican-themes
//使用主题，pelican的主题为一个主题一个目录，如果发现对应的主题的目录是空的，则直接去github里面找到相应主题，git下来，然后执行
pelican-themes -i bootstrap2

    对应在在pelicanconf.py中添加主题选择条目：

    //文件中找到这个字段，如果没有就添加一个
THEME = 'bootstrap2'

    在如下路径可以查看主题效果

    http://www.pelicanthemes.com/

    对于使用主题后pelicanconf.py的配置可以参考官方文档

    http://docs.getpelican.com/en/3.5.0/settings.html#themes

    如创建导航目录项：Menu Item设置。在你的博客中，可设置相应的菜单项，菜单项是通过pelicanconf.py设置的，具体如下所示：

    MENUITEMS = (("ITEM1","http://github.com"),
         ("ITEM2",URL),
        ......)
