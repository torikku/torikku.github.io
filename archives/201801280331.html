    博客回顾－－－MarkDown

    前面文章已经讲了怎么搭建静态页面网站，这里主要是介绍一下怎么写文章，当前用的是markdown解析器－－－MacDown工具

    首先需要安装MarkDown 

    //pip安装MarkDown
sudo pip install MarkDown

    安装完成后，就可以使用MarkDown语法了

    安装MacDown工具，这个是免费的，用的还不错，如果有其他的工具也可以使用

    该工具可以在github找到，直接下载应用包添加到电脑的application里面就可以了

    比如用该工具创建个about.md 页面

    在myBlog/content／pages里面用MacDown工具创建一个about.md文件，编辑内容如下

    Title: About Me        
Date: 2018-01-28

随便写点什么都可以！

    然后保存，直接发布，调测，就可以在127.0.0.1:8000看到效果啦。

    make publish
make serve

    如果要发布到github，则

    make github

    以下是编写md文件的一些关键字

    Title: Pelican+Github
Date: 2017-10-07 00:00
Modified: 2017-10-07 00:00
Tags: python, pelican
Slug: build-blog-system-by-pelican
Authors: torikku
Summary: blablablablablablablabla...
Status: draft

    其中前两似乎是必须要的，当前为发现是这样的，其他似乎可以不用。
